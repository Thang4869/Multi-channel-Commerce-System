## Key Workflows

### 1. Customer Order Journey
```
Browse Products → Add to Cart → Checkout → Payment → 
Order Confirmed → Warehouse Pick → Ship → Deliver → Complete
```
See: [order-flow.md](./order-flow.md)

### 2. Inventory Management
```
Supplier → Warehouse → Store → Sold → Reorder
```
See: [inventory-flow.md](./inventory-flow.md)

### 3. Delivery Process
```
Order Ready → Assign Shipper → Pick Up → In Transit → 
Deliver → Confirm Receipt
```
See: [delivery-flow.md](./delivery-flow.md)

### 4. User Authentication
```
Login → Verify Credentials → Issue JWT → Access Services → Refresh Token
```
See: [authentication-flow.md](./authentication-flow.md)
